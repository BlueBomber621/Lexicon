# UI art

Everything else, all `viewBox="0 0 48 48"`:

- **Tile bags** — `#icon-<bag>`: `icon-vowel`, `icon-consonant`, `icon-suffix`,
  `icon-latin`, `icon-science`, `icon-sorts`, `icon-peculiars`.
- **Sundries (slips)** — `#icon-slip-*`: `ink`, `overtime`, `purge`, `pulp`,
  `inkblot`, `vowel`, `sticker`, `typewriter`.
- **Pens** — `#icon-pen`, plus `icon-pen-double`, `icon-pen-highlighter`,
  `icon-pen-rough`.
- **Difficulty stamps** — `#icon-diff-*`: `note`, `letter`, `demy`, `royal`,
  `imperial`, `locked`.
- **Starting cases** — `#icon-deck-*`: `standard`, `pamphlet`, `gilded`,
  `archaic`, `jobbing`.

Ids come from `BAGS`, `CONSUMABLES`, `PENS`, `CFG.DIFFICULTIES`, and `DECKS`.
