# Sticker badges

Corner stickers that ride on Book covers, drawn from `#icon-sticker-<id>`.

- `viewBox="0 0 20 20"` — small; it sits at ~40% of the cover's width.
- File naming: `icon-sticker-<id>.svg`.
- Ids: `present`, `discount`, `signature`, `donated`, `bestseller`,
  `lamination` (see `STICKERS` in `js/content.js`).
